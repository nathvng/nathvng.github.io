Criei este portfólio para apresentar as atividades realizadas durante o curso técnico de Desenvolvimento de Sistemas no SENAI. Aqui, compartilho projetos, desafios e aprendizados adquiridos ao longo da formação, com o objetivo de demonstrar minhas habilidades e evolução na área.

Aprendizado:

Este portfólio foi desenvolvido com a orientação dos professores Leandro e Vedilson, durante o curso de Desenvolvimento de Sistemas no SENAI. Nos projetos, aprendemos a utilizar as linguagens HTML e CSS. O HTML (HyperText Markup Language) é fundamental para a estruturação de páginas web, permitindo criar elementos como títulos, parágrafos, links e imagens. Já o CSS (Cascading Style Sheets) é utilizado para estilizar esses elementos, proporcionando cores, fontes e layouts responsivos, tornando as páginas mais atraentes e funcionais.
